## Introduction

This tool allows to generate extension skeleton from WIDL file.

## Requirements

python-ply
python-jinja2 >= 2.7

## Usage

python WAPIOven.py -d <output directory> <webidl file>

ex) python WAPIOven.py -d ./tmp widls/datacontrol.widl
